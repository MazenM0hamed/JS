# 🐦 Twimba

A simple Twitter (X) clone built with vanilla JavaScript as part of the Scrimba Frontend Career Path.

Users can create tweets, like, retweet, and view replies. New tweets appear at the top of the feed and every tweet receives a unique UUID.

## 🚀 Features

- ✍️ Create a new tweet
- ❤️ Like and unlike tweets
- 🔁 Retweet and undo retweets
- 💬 Toggle replies
- 🆔 Unique UUID generated for every tweet
- 📌 New tweets appear at the top of the feed
- 🚫 Prevent empty tweets
- 🧹 Automatically clears the textarea after tweeting
- 📱 Responsive layout
