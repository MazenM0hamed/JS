# 🍔 Jimmy's Dinner - Restaurant Ordering App

A fully interactive restaurant ordering application built with vanilla JavaScript.  
Users can browse the menu, add items to their order, remove items, see the total price, complete their order, and enter payment details through a modal checkout experience.

This is my first major JavaScript project, where I practiced building a complete user flow from selecting products to completing a purchase.
## 🚀 Features

- Display restaurant menu items dynamically from JavaScript data
- Add items to the order
- Remove individual items from the order
- Automatically calculate the total price
- Dynamic order rendering
- Checkout modal with payment form
- Order completion message after payment
- Responsive and user-friendly interface
- Event delegation for handling dynamic buttons
