# 📌 Lead Tracker Chrome Extension

A simple Chrome extension that helps save and manage useful links/leads directly from your browser.

---

## 🚀 Features

- Save custom links manually
- Save the current browser tab with one click
- Store leads using localStorage
- Open saved links in a new tab
- Delete all saved leads with a double-click
- Persistent data after refreshing the browser

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chrome Extension API
- Local Storage
