# 🚨 We ❤️ Your Data - Modal Popup Project

A fun and interactive JavaScript project that simulates one of those annoying websites that tries to collect your data through a modal popup.

The project demonstrates DOM manipulation, event handling, form processing, timers, and dynamic content updates using vanilla JavaScript.

## 📸 Preview

The application displays a fake marketing page and, after a short delay, shows a modal asking the user to enter their name and email address.

Features include:

- Delayed modal appearance
- Accept and decline buttons
- Hover effect on the decline button
- Form validation
- Loading animation
- Dynamic content replacement
- Personalized message using user input
