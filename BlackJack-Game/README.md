# Blackjack Game 🃏

A simple Blackjack game built using JavaScript, HTML, and CSS.

## Features
- Start a new Blackjack game  
- Draw random cards  
- Track player cards and total sum  
- Detect Blackjack (21)  
- Detect when the player loses  
- Display player name and chips  

## JavaScript Concepts Used
- Objects  
- Arrays  
- Functions  
- Loops  
- Conditional statements  
- DOM manipulation
