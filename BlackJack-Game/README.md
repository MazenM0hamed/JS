# Blackjack Game 🎲🃏

A simple Blackjack game built with JavaScript, HTML, and CSS.
This project was created to practice JavaScript fundamentals such as:

-Objects
-Arrays
-Functions
-Loops
-Conditional statements
-DOM manipulation
-Features
-Start a new Blackjack game
-Draw random cards
-Track player cards and total sum
-Detect Blackjack (21)
-Detect when the player loses
-Display player name and chips
